mailx 发送邮件小计

发表于 2017-03-21 | 更新于 2018-03-15 | 分类于 监控

安装postfix服务

yum install postfix
/etc/init.d/postfix start
chkconfig postfix on

发送格式

mailx -s subject user@xxx.com < message_file
echo “内容” | mailx -s “邮件标题” user@xxx.com

其他选项

-r 指定发件人
-c 指定抄送人
-b 指定密送人
多个收件人使用逗号分隔
有些邮箱（如qq邮箱）会当成垃圾邮件拦截，日志中可以看到500错误
