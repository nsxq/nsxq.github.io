数据库报错信息

该错误原因为表空间无法扩展故障，查找后得知单个表空间文件最大32G，有两个方法扩大表空间
（1）创建多个数据文件，都不能超过32g
（2）创建大表空间。create bigfile tablespace 他的上限是8EB，不过，oracle10及以后版本才能用。
